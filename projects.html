Projects

Carbon Trading Component:

Engineered and implemented a Java-based software component to model carbon credit trading, utilizing object-oriented programming (OOP) principles and data structures (Map & List) to create a modular system with kernel and secondary methods for tracking emissions, purchasing credits, and calculating limit penalties. Simulated real-world carbon trading scenarios by incorporating environmental economics concepts such as carbon pricing, emission limits, and offset strategies, and validated the model’s accuracy through test cases.

Natural Number Calculator:

Implemented a natural number calculator using Java and Swing framework, following the Model-View-Controller design pattern to create a modular and scalable application where the model handled arithmetic logic, the view managed the graphical interface, and the controller facilitated communication between the two. Developed controller logic to handle arithmetic operations (addition, subtraction, multiplication, division, power, and root) on unbounded natural numbers, ensuring robustness by implementing input validation and error handling to prevent crashes from invalid inputs, such as division by zero or invalid root calculations, while maintaining precise calculations for extremely large numbers. Built a dynamic and interactive GUI using Swing components, including buttons for digits and operations and text areas for input and output displays, designed to dynamically enable/disable operations based on the current state of the model (e.g., disabling division when the divisor is zero) and provide real-time updates to the view to reflect changes in the model, ensuring a seamless and intuitive user experience. Utilized JUnit for comprehensive unit testing of the model and controller components, validating the accuracy of arithmetic operations and state management, while addressing edge cases like stack overflow for extremely large numbers and recursive operation limits through debugging, optimization, and collaboration with peers to improve the application’s reliability and performance.

Human Genome String Reassembly:

Created a Java-based algorithm to reassemble fragmented strings, inspired by DNA sequence reconstruction techniques from the Human Genome Project by utilizing Set data structures and greedy optimization to efficiently merge overlapping fragments, optimizing for performance and scalability. Employed test-driven development (TDD) with JUnit to validate functionality, ensure robustness and scalability for large datasets (1000+ fragments), and to validate the handling of empty or non-overlapping string fragments, as well as integration tests to verify the correctness of the various method in reconstructing fragmented text. Applied computational biology concepts and leveraged object-oriented programming (OOP) to handle edge cases, custom delimiters, and large-scale text reconstruction, simulating real-world genome reassembly challenges.
